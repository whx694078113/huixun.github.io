数值转换 parseInt 遇到空字符串转化为NaN 如果第一个是字符串 将会显示到非空字符串结束 小数点并不是有效地数字字符

  • var num1 = parseInt('');
  • console.log(num1) //Nan
  • var num2 = parseInt('123sss');
  • console.log(num2) //123
  • var num3 = parseInt('22.3');
  • console.log(num3) //22
  • var num4 = parseInt("70",8);
  • console.log(num4) //56
  • var num5 = parseInt("70",10);
  • console.log(num5) //70
  • var num6 = parseInt("0xf");
  • console.log(num6) //15
